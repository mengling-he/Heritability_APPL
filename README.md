# Heritability_APPL


## About
This repository contains code to perform heritability analyses using the APPL image phenotypes. 
The notebooks demonstrate data preparation, model fitting, and result visualization for reproducible analyses. 

## Content
1. Analysis/ — directory (contains serveral analysis steps, there is 'data','script' and 'result' in each step folder)
2. Code/ — directory (function for LMM model).
3. raw_data/ — directory (provided by PlantCV).
